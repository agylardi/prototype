Simple Price Calculator is a jQuery plugin that can transform any html based form into a price calculation form. You can use it to provide instant price quotes or estimates on products and services for your visitors. A total with details can be displayed dynamically within your page.

Form Examples

Fully coded Examples on how to implement are provided in the examples folder.  If you are having trouble with the instructions below, please use one of the templates in the examples folder to get started. The ajax example is the only one that provides code to e-mail the form via the emailform.php file. This is a for example purposes only. If you need to use this functionality for your form, it is recommended that you have programming knowledge or hire a programmer to customize this functionality for your form.

Quick Start Guide

1. Download the zip file and extract the contents. Place the js & css folder with your html files (or any folder of your choosing).

2. Copy and paste the code below and add to your html file:

<script src="/js/jquery-3.3.1.min.js"></script>
<script src="js/jquery.number.min.js"></script>
<script src="js/simplepricecalc.js"></script>
<link rel="stylesheet" href="css/spcstyle.css" type="text/css"  />

3. Copy and paste the SimplePriceCalc function to your html file. Replace the formid text with the id of your form.

$(document).ready( function ($) {
$("#formid").SimplePriceCalc();
});


4. Add a data-price attribute to the input or option tags within your form that you need computed with price value. 

For example:

Change   <input type="text" />  to:  <input type="text" data-price="5" />

5. You can also add a data-label attribute to any tag if you want the calculation and text displayed in the details section.

For example:

Change   <input type="text" />  to:  <input type="text" data-price="5" data-label="Fruit Cost"/>
 

Detailed Guide


The attributes below to add extra options to the form elements:

data-price- determines value of the item(for option and input text tags) to add to total. If blank, nothing will be displayed.
*data-price attribute is not used on checkbox or radio inputs. Instead, the value attribute can be used.

ex: <input type="text" data-price="5" />

data-label- Displays a label along with pricing or custom text in the total box for html element if selected

ex: <input type="checkbox" data-label="Product" value="5"/> would display Product: $5 in details box if checked

data-mult- multiplies the entire total by the value in this field. Typically used if you want to add an input box to multiply the total price of the form.

ex: <input type="text" data-mult="1" data-label="Amount of websites needed" />   
If the total is $500 before a number is entered and 5 is entered in this box, the total will then change to $2500

data-mult-ids- Similar to data mult but this function multiplies only the values of the id's you specify in this field. Typically used if you want to add an input to calculate the price for specific items.

ex: <input type="text" data-mult-ids="widget1,widget2" />   
If the total is $500 before a number is entered and the values of widget1 = 100 and widget2= 50, they will be multiplied by the value entered in this box. If that number is 2, then the total will then change to $800 ($500+ (150x2))

data-merge- multiplies the input value of multiple id's by the data-price in this element. Each id must be separated by a comma and they will be added and then multiplied by the elements data-price.

ex: The following takes the value inputted in the features and features 2 box and multiplies it by the data-price value of 10 for the total sqft box.

<h4> Width: </h4> <input type="text" data-price="0" id="features" data-label="width" />
<h4> Length: </h4> <input type="text" data-price="0" id="features2" data-label="height"/>
<h4> Total SQFT: </h4> <input type="text" data-price="10" data-merge="features,features2" data-label="Total SQFT" disabled />

Some attributes can be combined as well:

<option data-price="500" data-label="Basic Website"> Basic Website </option>


How to change styling of total box


Add the fixed-total class to your form attribute in order to have the total affixed to the side. Without this specified, the total will display at the bottom of the form.

<form id="quoteform" class="fixed-total">

* Make sure your html file is linking to the spcstyle.css sheet if having trouble

Simple Price Calculator Settings

The plugin settings give you the option to change some of the text or currency from within your form. Just add the parameters below to your SimplePriceCalc function:

$(document).ready( function ($) {
  $("#formid").SimplePriceCalc({
  currency:'$',   // Place any symbol of your choosing here, default is the dollar sign
  detailslabel: 'Details:' ,   // Change the details text
  totallabel: 'Total:'   // Change the total text
  });
});
